# create-vue-microfrontends

## 项目简述

> 项目是一个 Single-spa+Vue 框架搭建的微前端项目

## 具体介绍

[从零搭建一个微前端项目(Single-spa)](https://www.jianshu.com/p/d00f835050d8)

## 使用步骤

1.主应用(app-main)

```js
npm install
npm run serve
```

2.子应用(app-demo1/app-demo2)

```js
npm install
npm run serve
```
